# Title : Hotel-Booking-Analysis

## Project Overview

This project analyzes hotel booking data to understand customer booking behavior and business performance. The dashboard was created in Microsoft Excel using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

  ## Data Description

  The dataset contains hotel booking information for both City Hotels and Resort Hotels. It includes booking details, customer information, reservation status, meal type, market segment, country, arrival date, length of stay, and other booking-related attributes. The dataset consists of 119,390 records and 32 columns.

  ## Data Preparation

  * Checked the dataset for missing values, duplicate records, and outliers
  * Removed invalid records where necessary
  * Created a new Arrival Date column by combining the day, month, and year columns
  * Prepared the dataset for analysis using Pivot Tables and dashboard creation

    ## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* KPI Cards
* Slicers

  ## Dashboard

  The interactive dashboard provides a clear overview of hotel booking performance using KPI Cards, Pivot Charts, and Slicers. It helps users analyze booking trends, customer behavior, cancellation patterns, hotel performance, meal preferences, and market segments through interactive visualizations.

  ## Key Insights

  * City Hotels received more bookings than Resort Hotels
  * August had the highest number of bookings
  * Portugal had the highest number of bookings
  * The cancellation rate was 37.04%
  * BB was the most popular meal type
  * Online TA contributed the highest number of bookings
  * Most customers were Transient customers
